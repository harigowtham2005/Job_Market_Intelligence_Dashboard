# 📊 Job Market Intelligence Dashboard

## Overview

An end-to-end Data Analytics project that analyzes job market trends using Python, MySQL, and Power BI.

The project provides insights into:

- Hiring trends
- Salary trends
- Career opportunities
- Fresher opportunities
- Market competition

---

## Tech Stack

- Python
- Pandas
- Matplotlib
- MySQL
- Power BI

---

## Dataset

- 5,000 Job Postings
- 53 Companies
- 17 Cities
- 4,817 Skills

---

## Dashboard Pages

### 📊 Executive Overview
High-level job market KPIs and trends.

### 🏢 Hiring Intelligence
Top hiring companies, cities, company types, and location analysis.

### 💰 Salary Intelligence
Salary trends across industries, job roles, and experience levels.

### 🎯 Career Intelligence
Demanded skills, education requirements, competition ratio, and popular job roles.

### 🎓 Fresher Insights
Entry-level opportunities, recruiter analysis, salary insights, and competition.

---

## Project Workflow

CSV Dataset
→ Python Data Cleaning
→ Exploratory Data Analysis
→ MySQL Database
→ SQL Business Analysis
→ Power BI Dashboard
→ Business Insights
